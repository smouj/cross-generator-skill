# Generador de Análisis

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad de Generador de Análisis para OpenClaw.

##Qué Resuelve
Generador impulsado por IA para tareas de análisis.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales listos para producción
- Enfoque de seguridad primero
- Soporte para revertir cambios

## Uso
/cross-generator

## Ejemplo
```
Analyze code for analysis issues.
```

## Instalación
Esta habilidad está automáticamente disponible en OpenClaw.
